# CaeruAI — コンテンツ企画メーカー

> *Caeru* — ラテン語で「天空の青」。アイデアの可能性は無限大。

## どういうサービスか

今日の気分を一言入力するだけで、あなたのプロフィールと過去のフィードバックを学習したAIが「あなた専用の企画」を5つ生成します。汎用的な提案ではなく、あなたのスタイル・ターゲット・こだわりに合わせた企画だけを出力します。

ChatGPTとの違い：CaeruAIはあなたのチャンネルの文脈を知っています。生成後に別のAIが品質を批評して基準を満たさない企画だけを書き直す2段階パイプラインを採用しています。

---

## 機能

- **企画生成** — 気分・プロフィール・過去フィードバックを元に企画を5件提案
- **2段階AIパイプライン** — 生成 → 批評 → 改善で品質を担保
- **マルチプラットフォーム対応** — YouTube / TikTok・Reels / Instagram / Podcast / ブログ・note / X / 配信
- **プログレッシブセットアップ** — 2問答えればすぐ使える。追加回答するほどAI精度が上がる
- **フィードバック学習** — 「いい感じ」「違う」の評価が次回生成に反映される
- **履歴管理** — 過去の生成結果をいつでも確認
- **ダークモード対応**
